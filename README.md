# Prodly

To setup

Do npm install in both frontend and electron directories

Make a .env file and put GROQ_API_KEY="your_api_key_here" get your api key from groq site

Make a virtual environment with python -m venv .venv

Then pip install -r requirements.txt to virtual environment

To run

In 3 terminals electron frontend and backend 

Start frontened and electron with npm run dev and npm start respectively

Start backend by running python main.py
